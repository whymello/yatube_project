# yatube_project

## Описание

Yatube — это блог-платформа и социальная сеть, где пользователи могут публиковать посты, подписываться на авторов и взаимодействовать с контентом.

Проект находится в стадии активной разработки и будет дополняться по мере его развития.

## Функциональность

- **Административная панель Django**:

  - создание и редактирование пользователей и групп;
  - создание, редактирование и удаление постов;
  - привязка постов к группам при создании или редактировании.

- **Пользовательская часть**:

  - главная страница со списком постов (с постраничной навигацией);
  - страница группы с постами этой группы;
  - профили пользователей с их постами;
  - детальный просмотр поста;
  - создание и редактирование постов доступно только авторизованному пользователю;
  - адаптивное навигационное меню, которое изменяется в зависимости от того, авторизован пользователь или нет.

- **Регистрация и авторизация**:

  - регистрация нового пользователя;
  - авторизация и выход из системы;
  - смена и сброс пароля.

- **Шаблоны и интерфейс**:

  - базовый шаблон `base.html`, включающий `header.html` и `footer.html`;
  - текущий год в футере отображается с помощью контекст-процессора `year`;
  - адаптивное меню реализовано с использованием JS и Bootstrap.

- **Дополнительно**:

  - постраничная навигация реализована через объект `Paginator`;
  - эмуляция почтового сервера через `EmailBackend`, письма сохраняются в директории `sent_emails`.

- **Статические страницы**:

  - об авторе;
  - технологии.

## Технологии

- Python 3.14
- Django 6.0
- SQLite  
- Bootstrap, JavaScript (для адаптивного меню)

## Структура проекта

- **Приложения**:

  - `posts` — управление постами и группами;
  - `users` — регистрация, авторизация, профили пользователей;
  - `core` — пользовательские фильтры и вспомогательные функции;
  - `about` — статические страницы «об авторе» и «технологии».

- **Шаблоны**:

  - `base.html`, `header.html`, `footer.html`;
  - `index.html`, `group_list.html`, `profile.html`, `post_detail.html`, `create_post.html` и др.

- **Статика**:

  - CSS, JS и изображения лежат в папке `static/`.

## Запуск проекта в dev-режиме

1. Клонируйте репозиторий:

    ```bash
    git clone https://github.com/whymello/yatube_project.git
    ```

2. Перейдите в папку проекта и создайте виртуальное окружение:

    ```bash
    python -m venv venv
    source venv/bin/activate  # для GNU/Linux, macOS
    venv\Scripts\activate     # для Windows
    ```

3. Установите зависимости:

    ```bash
    pip install -r requirements.txt
    ```

4. Выполните миграции:

    ```bash
    python manage.py migrate
    ```

5. Создайте суперпользователя (по желанию):

    ```bash
    python manage.py createsuperuser
    ```

6. Запустите сервер разработки:

    ```bash
    python manage.py runserver
    ```

    Проект будет доступен по адресу: <http://127.0.0.1:8000/>
