# Yatube

## Описание

Yatube — это блог-платформа и социальная сеть, где пользователи могут публиковать посты, подписываться на авторов, комментировать записи и взаимодействовать с контентом.

Проект представляет собой функционально завершённое веб-приложение с серверной HTML-частью и REST API.

Дальнейшее развитие проекта возможно по мере появления новых идей и изучения дополнительных технологий.

## Демо

Онлайн-версия проекта доступна по адресу: <https://whymello.pythonanywhere.com>

## Функциональность

### Веб-интерфейс

- Главная страница со списком постов (с пагинацией);
- Фильтрация: все посты / посты избранных авторов;
- Страницы групп;
- Профили пользователей;
- Детальная страница поста;
- Создание, редактирование и удаление постов доступно авторизованным пользователям;
- Загрузка изображений к постам (через `sorl.thumbnail`);
- Комментирование постов доступно авторизованным пользователям;
- Подписка / отписка на авторов;
- Адаптивное меню (Bootstrap);
- Кастомные страницы ошибок: 403, 403csrf, 404, 500;
- Кеширование списка постов на главной странице;
- Оптимизация запросов к БД.

### Регистрация и авторизация

- Регистрация пользователя;
- Авторизация и выход;
- Смена и сброс пароля;
- Эмуляция почтового сервера (письма сохраняются в `sent_emails`).

## REST API

API реализован с использованием Django REST framework.

### Доступные ресурсы

- **Posts** — CRUD;
- **Comments** — CRUD;
- **Groups** — список и детальное представление;
- **Follow** — просмотр подписок и оформление подписки.

### Аутентификация

- JWT-аутентификация;
- Реализована с использованием `djoser`.

### Дополнительно

- Ограничения прав доступа;
- Throttling (ежедневные и ежеминутные лимиты запросов);
- Поддержка CORS;
- Автоматическая генерация OpenAPI-документации.

Документация API доступна после запуска проекта (путь зависит от настроек, например `/api/swagger/ui/` и `/api/redoc/`).

## Структура проекта

### Приложения

- `posts` — посты, группы, комментарии, подписки;
- `users` — регистрация и авторизация;
- `core` — пользовательские фильтры и вспомогательные функции;
- `about` — статические страницы.

## Тестирование

Реализованы автоматические тесты с использованием встроенного тестового фреймворка Django.

Покрытие тестами:

- `about`: views, urls;
- `users`: views, urls, forms;
- `posts`: views, urls, forms, models, загрузка изображений.

Для запуска тестов выполните команду:

```bash
python manage.py test
```

## Запуск проекта в dev-режиме

1. Клонируйте репозиторий:

    ```bash
    git clone https://github.com/whymello/yatube_project.git
    ```

2. Перейдите в папку проекта и создайте виртуальное окружение:

    ```bash
    python -m venv venv
    source venv/bin/activate  # для GNU/Linux, macOS
    venv\Scripts\activate     # для Windows
    ```

3. Установите зависимости:

    ```bash
    pip install -r requirements.txt
    ```

4. Перейдите в папку `yatube` и выполните миграции:

    ```bash
    python manage.py migrate
    ```

5. Создайте суперпользователя (по желанию):

    ```bash
    python manage.py createsuperuser
    ```

6. Запустите сервер разработки:

    ```bash
    python manage.py runserver
    ```

    Проект будет доступен по адресу: <http://127.0.0.1:8000/>
