Технологии и подходы 🛠

  • 🐍 Python
  • 🌐 Django
  • 🗄 Django ORM + SQLite3
  • 🎨 Bootstrap

Этот проект был создан с использованием классического и проверенного стека для Django-приложений.

В качестве основного языка используется Python — читаемый, гибкий и отлично подходящий для быстрого прототипирования и разработки веб-сервисов. Фреймворк Django помог выстроить чёткую архитектуру проекта и не изобретать велосипед там, где он уже отлично крутится.

В проекте активно применялись маршруты, шаблоны и ORM — это позволило сосредоточиться на логике приложения, не погружаясь слишком глубоко в низкоуровневую работу с базой данных.

Для внешнего вида использовался Bootstrap — он помог быстро сверстать аккуратные страницы, которые нормально выглядят как на компьютере, так и на мобильных устройствах.

Иногда код не работает с первого раза. Иногда — со второго. Но в итоге он всегда начинает работать… особенно если рядом есть уточка 🦆
